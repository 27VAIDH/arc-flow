## Codebase Patterns
- CSS theme variables defined in index.css @theme block AND duplicated in tailwind.config.ts arc colors — both must be updated together
- Dark mode uses @variant dark (&:is(.dark, .dark *)) pattern in index.css
- Section headers pattern: text-[11px] font-medium/font-semibold + uppercase tracking-wider — used in PinnedAppsRow, FolderTree, App.tsx tab count, SettingsPanel
- All section separators use: border-t border-gray-200/80 dark:border-arc-border or border-b variant
- applyPanelColor() in useTheme.ts sets --color-arc-panel-bg CSS variable — exported for SettingsPanel direct use
- Panel color ownership is in App.tsx (not useTheme) for per-workspace override support
- DraggableTabItem active state uses border-l-[3px] border-l-arc-accent + font-medium + dark:bg-arc-surface
- Pinned app icons: w-9 h-9 rounded-full dark:bg-arc-surface with group-hover:scale-105 and ring-2 active indicator
- Search input: border border-transparent focus:border-arc-accent/50 focus:ring-1 focus:ring-arc-accent/30 shadow-inner
- Footer has 3 rows: workspace switcher, suspended count text, Focus+Settings buttons
- WorkspaceSwitcher.tsx contains workspace emoji buttons, +button, emoji/color pickers, dot indicators, arrow buttons
- Folder icon uses text-arc-accent dark:text-arc-accent-hover for coloring
- .backdrop-frosted class exists in index.css but is not applied to any element currently
- Transition classes: transition-colors duration-150 and transition-all duration-150 used throughout
